# 说明

日程管理后台，数据库课程项目。

有一个使用OpenGauss的分支，可供需要使用OpenGauss做校内项目的同学参考。

slave分支为spark集群+Open Gauss版本

# 如何运行

克隆到本地，在pom.xml加载项目，根据application_ref和generator_ref自己写一份application.yml和generator.xml即可，数据库的结构存放在resource目录下的SQL文件中。
